# 小程序一级推广分阶段奖励功能需求设计文档

## 一、文档概述

### 1.1 需求背景

为提升小程序新用户拉新效率、促进用户转化成交，平台新增**一级推广分阶段奖励**功能。推广用户通过专属推广渠道邀请新用户，分为「注册登录奖励」和「交易完成奖励」两个阶段发放收益，所有奖励自动入账用户小程序余额，可参与后续提现，助力平台用户增长与订单增量。

### 1.2 核心需求规则

- 推广层级：仅支持**一级推广**，无二级及多级分佣，合规无传销风险

- 阶段一奖励：新用户通过推广渠道**完成微信授权注册登录**，推广人获得 **2元** 奖励

- 阶段二奖励：该新用户后续完成**买卡/卖卡有效交易订单**（订单完结无退款），推广人再获得 **8元** 奖励

- 单用户总奖励：成功拉新并促成交易，推广人合计可得 **10元/人**

- 奖励发放形式：所有奖励自动发放至推广人**小程序个人账户余额**

### 1.3 合规说明

- 仅一级推广模式，无层级裂变、无入门费、无强制分享拉新，完全符合微信小程序运营规范

- 奖励分为注册轻奖励+交易奖励，兼顾拉新与真实交易，无纯刷单套利漏洞

- 无需特殊资质、无需额外备案，依托现有个体户主体、ICP备案即可正常上线审核

## 二、角色定义

- **推广人**：已完成小程序授权登录的存量用户，拥有专属推广码/推广链接，可分享拉新

- **被推广新用户**：从未在本小程序注册、登录、留存记录的全新用户

- **平台管理员**：后台配置奖励规则、查看推广数据、处理异常奖励、风控作弊账号

## 三、核心功能详细需求

### 3.1 推广中心前端模块

入口：小程序个人中心新增【我的推广】固定入口

页面核心展示内容：

- 专属永久推广二维码、专属微信分享短链接

- 清晰的分阶段奖励规则公示（注册得2元、成交再得8元）

- 数据统计模块：累计推广人数、累计总奖励、待入账奖励、已入账奖励

- 推广记录列表：展示新用户昵称、注册时间、绑定状态、注册奖励状态、交易订单状态、成交奖励状态、对应奖励金额

- 余额明细入口：可查看所有推广奖励入账记录

### 3.2 新用户绑定逻辑

1. 新用户通过推广二维码/推广链接进入小程序，系统自动抓取推广人唯一ID

2. 新用户完成微信授权登录、注册账号后，系统**自动绑定一级推广关系**

3. 绑定规则：永久绑定，同一新用户仅绑定首次推广人，不支持二次换绑

4. 老用户、已绑定推广关系的用户，不再重复绑定、不重复发放新人奖励

### 3.3 分阶段奖励发放核心逻辑

#### 3.3.1 第一阶段：注册登录奖励（2元）

**触发条件**：新用户首次通过推广渠道进入小程序，完成微信授权、注册登录，生成用户账号

**发放规则**：

- 满足条件后**实时自动发放**2元奖励至推广人余额

- 入账备注：「新用户注册推广奖励」

- 单用户仅发放一次，重复注册不重复奖励

#### 3.3.2 第二阶段：交易完成奖励（8元）

**触发条件（全部满足）**：

- 新用户已绑定对应推广人

- 新用户在平台产生**买卡/卖卡交易订单**

- 订单状态变更为【交易完成】（核销成功、结算完成、无退款、无取消、无纠纷）

**发放规则**：

- 订单完成后**实时自动发放**8元奖励至推广人余额

- 入账备注：「新用户交易推广奖励」

- 同一新用户仅首次有效成交订单发放一次8元奖励，后续复购不再奖励

### 3.4 余额联动规则

- 两阶段推广奖励全部计入用户**可提现余额**，与交易收益余额通用，无使用限制

- 余额明细区分展示两种推广奖励，账目独立清晰，可追溯

- 支持平台统一提现规则，可绑定银行卡自主提现

### 3.5 消息通知机制

两个阶段奖励到账后，小程序自动推送服务通知：

- 注册奖励到账通知：您邀请的新用户已注册，获得推广奖励2元

- 交易奖励到账通知：您邀请的新用户已完成交易，获得推广奖励8元

### 3.6 后台管理功能

#### 3.6.1 奖励参数配置

- 后台可自定义修改注册奖励金额、交易奖励金额

- 支持一键开启/关闭推广奖励活动

- 可配置奖励发放延迟时间、风控拦截开关

#### 3.6.2 数据统计与明细

- 整体数据：总推广人数、注册奖励总支出、交易奖励总支出、总推广成本

- 推广明细：推广人信息、新用户信息、注册时间、注册奖励状态、订单编号、交易完成时间、交易奖励状态、发放时间

- 支持数据筛选、查询、导出、订单溯源

#### 3.6.3 异常管理

- 支持手动补发、冻结、收回异常奖励

- 可查看作弊、退款、无效订单对应的奖励记录

## 四、异常场景与风控规则

### 4.1 不发放奖励场景

- 新用户为平台老用户、已注册用户，重复进入不发注册奖励

- 新用户注册后未产生任何交易，无8元交易奖励

- 新用户订单取消、退款、交易失败、违规拦截，不发放交易奖励

- 一机多号、批量注册、恶意刷单、作弊刷量账号，系统拦截所有奖励

### 4.2 奖励回滚机制

若奖励发放后，新用户订单出现退款、纠纷撤销等异常情况，后台支持手动收回对应8元交易奖励，从推广人余额中抵扣，留存操作日志，保证平台资金合规。

### 4.3 防作弊规则

- 基于微信OpenID、设备唯一标识双重校验，防止批量注册刷奖

- 严格区分新老用户，杜绝老用户换号套利

- 异常高频推广、批量同款设备注册，自动标记风控

## 五、完整业务流程

1. 推广人进入【我的推广】页面，获取专属推广码/链接并分享

2. 新用户扫码/点击链接进入小程序，自动绑定推广关系

3. 新用户完成注册登录 → 推广人自动到账2元奖励

4. 新用户在平台完成买卡/卖卡交易，订单最终完成

5. 订单完成后 → 推广人自动到账8元奖励

6. 推广人可在余额明细、推广记录中查看全部收益，支持提现

## 六、前端公示规则文案（可直接上架）

**一级推广奖励活动规则**

1. 本活动为平台一级推广活动，仅存在一级推广关系，无多级裂变收益；

2. 新用户定义：从未在本小程序注册、登录、产生交易的全新用户；

3. 奖励规则：成功邀请新用户完成注册登录，可获得2元推广奖励；被邀请新用户完成平台卡券买/卖有效交易订单且订单完结无退款，可额外获得8元推广奖励；

4. 单名有效新用户最多可为推广人带来合计10元推广收益，同一用户仅首次注册、首次成交可分别触发一次奖励；

5. 所有推广奖励自动发放至用户小程序个人账户余额，可参与平台提现；

6. 严禁批量注册、刷单、作弊刷奖等违规行为，平台有权冻结、收回违规奖励并封禁账号；

7. 平台保留活动规则最终解释权，规则调整将在小程序内公示告知。
