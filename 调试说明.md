## 如何运行小程序

### 方法一：使用微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择目录 `D:\福利收回`
3. AppID 已配置为 `wx8e0a92186bf86b07`（可使用测试号）
4. 点击"编译"按钮运行

### 方法二：命令行运行

```bash

cd D:\福利收回\miniprogram

# 确保依赖已安装
npm install

# 在微信开发者工具中构建npm
# 工具 -> 构建npm
```

## 注意事项

1. **模拟数据模式**：首页 `index.js` 中设置了 `USE_MOCK_DATA = true`，使用模拟数据进行开发测试。后端服务启动后可改为 `false`。

2. **后端服务**：项目包含后端服务代码（server目录），需要配置MySQL和Redis才能完整运行。如仅测试前端，可使用模拟数据。

3. **Vant Weapp组件**：已安装 `@vant/weapp@1.11.7`，确保在微信开发者工具中执行"构建npm"。

4. **TabBar图标**：TabBar使用的图标文件已存在于 `assets/icons/` 目录。

## 功能说明

### 首页
- 展示卡券分类网格
- 热门回收推荐
- 搜索卡券类型
- 下拉刷新

### 登录页
- 微信一键登录
- 手机号登录
- 用户协议和隐私政策

### 实名认证页
- 输入真实姓名和身份证号
- 表单验证
- 认证协议

### 卡券详情页
- 选择面值
- 输入卡号卡密
- 计算回收金额
- 提交订单

### 订单列表页
- Tab切换不同状态
- 下拉刷新和上拉加载
- 订单卡片展示

### 个人中心页
- 用户信息展示
- 账户余额
- 功能菜单
- 退出登录

### 提现页
- 显示账户余额
- 输入提现金额
- 选择结算账户

### 结算账户页

- 添加/删除银行卡
- 设置默认账户
