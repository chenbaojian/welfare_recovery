# 联系客服功能设计文档

## 一、功能概述

### 1.1 功能背景
当前"联系客服"功能仅使用 `wx.makePhoneCall` 拨打占位电话号码（`400-xxx-xxxx`），无法真正为用户提供客服服务。需要设计一个完整的联系客服功能，支持多种客服渠道，提升用户服务体验。

### 1.2 功能目标
- 提供多渠道客服联系方式（在线客服、电话客服、微信客服）
- 展示常见问题FAQ，减少客服压力
- 支持用户留言反馈，异步处理问题
- 保障客服信息可配置、可维护

### 1.3 目标用户
所有小程序用户（游客、普通用户、认证用户均可使用，无需登录门槛）

### 2.3 交互流程

#### 2.3.1 在线客服流程

```
用户点击"立即咨询"
    │
    ▼
触发微信客服消息能力（button open-type="contact")
    │
    ▼
打开微信客服对话界面
    │
    ▼
用户与客服实时沟通
```

#### 2.3.2 电话客服流程

```
用户点击"拨打电话"
    │
    ▼
调用 wx.makePhoneCall({ phoneNumber: '400-888-8888' })
    │
    ├─ 成功 → 拨号界面
    │
    └─ 失败 → Toast提示"拨打电话失败，请尝试其他方式"
```

#### 2.3.3 微信客服号流程

```
用户点击"复制微信号"
    │
    ▼
调用 wx.setClipboardData({ data: 'FLHS_kefu' })
    │
    ▼
Toast提示"微信号已复制，请在微信中搜索添加"
    │
    ▼
用户手动在微信中搜索添加客服号
```

#### 2.3.4 FAQ查看流程

```
用户点击某个FAQ问题
    │
    ▼
展开/折叠显示答案内容
    │
    ▼
用户查看答案
```

#### 2.3.5 留言反馈流程

```
用户填写留言信息
    │
    ├─ 选择问题分类（必选）
    ├─ 输入问题描述（必填，最少10字）
    ├─ 输入联系方式（必填）
    │
    ▼
点击"提交留言"
    │
    ├─ 校验失败 → 提示具体错误
    │
    ▼
校验通过，提交到后端
    │
    ├─ 成功 → Toast提示"留言已提交，客服将在24小时内回复"
    │
    └─ 失败 → Toast提示"提交失败，请稍后重试"
```

## 四、接口设计

### 4.1 提交留言反馈

**接口路径**：`POST /common/feedback`

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| category | string | 是 | 问题分类 |
| content | string | 是 | 问题描述（≥10字） |
| contact | string | 是 | 联系方式 |
| userId | string | 否 | 用户ID |

**响应数据**：

```json
{
  "code": 200,
  "message": "留言已提交",
  "data": {
    "id": "FB202401010001",
    "status": "pending"
  }
}
```

### 5.3 关键技术要点

1. **在线客服**：使用微信小程序 `<button open-type="contact">` 组件，需在微信后台配置客服账号
2. **电话客服**：使用 `wx.makePhoneCall` API
3. **微信客服号**：使用 `wx.setClipboardData` 复制微信号到剪贴板
4. **FAQ折叠展开**：使用 Vant Weapp 的 `van-collapse` / `van-collapse-item` 组件
5. **留言分类选择**：使用 Vant Weapp 的 `van-picker` + `van-popup` 组件
6. **表单校验**：问题描述最少10字，联系方式必填

## 七、与现有模块的集成

### 7.1 个人中心页面修改

将 `mine.js` 中的 `onContactService` 方法从直接拨打电话改为跳转到联系客服页面：

```javascript
// 修改前
onContactService() {
  wx.makePhoneCall({
    phoneNumber: '400-xxx-xxxx',
    fail: () => {
      wx.showToast({ title: '拨打电话失败', icon: 'none' });
    }
  });
}

// 修改后
onContactService() {
  wx.navigateTo({
    url: '/pages/contact/contact'
  });
}
```

### 7.2 app.json 路由注册

在 `app.json` 的 `pages` 数组中新增：

```json
"pages/contact/contact"
```
